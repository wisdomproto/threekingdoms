## 0. 핵심 성격

- **1회차 = 원작 영걸전 맵/적 배치 95%+ 동일**(우리 §2-9 "맵=영걸전" 재현과 정합). 2회차+ 적 증량·잡졸
  열전 특화, 3회차 "지옥"은 적이 네임드급 능력치 + 보조템 기본 장착.
- **세이브&로드 신공 불가 = 시드 고정 확률.** 같은 상황 반복 시 같은 결과(맞으면 계속 맞음), 변수를
  바꾸면 결과가 갈림. ⭐ **우리가 채택한 "시드 고정 RNG" 모델의 직접 선례** — 순수 랜덤이 아니라
  (게임상태+시드) 결정. 리플레이/리더보드/밸런스 sim과 공존 가능한 이유.
- 성장: **공훈 모드 / 열매 모드 택1**(제갈량 등용 후 공훈 재분배 가능). 회차 계승. 병종 변경은 유비만.
- 우군(아군 조작 불가, 특정전투 자동참전) 분리 — 일부는 조건 충족 시 아군화. (우리 §6 게스트/우군 정합)

## 2. 상태이상 서브시스템 (별도 레이어 — 현재 우리 0)

- 종류: **중독**(지속 피해) · **혼란**(자행동/오발) · **금책**(책략 봉인) · **부동**(이동 봉인) ·
  사기저하 · 방어/순발/이동 등 스탯 하강.
- 부여 = **확률**(예: 75% 혼란) 또는 **확정**. 매턴 처리 + 회복(상태회복 책략/아이템).
- 반사/되돌림(항마방패)·면역(병종/아이템)과 상호작용.

## 4. 보물·아이템 → 효과 (대표 — §10 도감 소재)

적토마=돌파+선제 / 거울갑옷=간접반사 / 연환갑옷=재반격 / 사소검=필중·고정피해 / 방천화극=부동+흡혈 /
조황비전=연속행동(매턴 추가행동) / 신수장갑=명중+25%·반격 풀데미지 / 옥새=매턴 전능력상승 /
손자병법서=전 공격 회심 / 의천검=일치단결 / 백은갑옷=책략피해 −40% / 봉황깃옷=최대HP+.
획득 = 상점·보물고·일기토·대화 이벤트(우리 §10 보물 3문법과 정합). **보물 조합 시너지**가 빌드의 핵
(예: 청룡언월도+적토마+거울갑옷 = 근접 물리 완봉).

## 5. 우리 이식 시 가드레일 / 차이

- **시드 고정**(세이브스컴 방지 = 레퍼런스 일치). 리플레이=시드 기록, 밸런스 sim=N시드 분포.
