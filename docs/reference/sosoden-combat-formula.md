## 1. 능력치 파생 (장수 능력 → 부대 능력)

| 부대 능력 | ← 장수 능력 | 영향 |
|---|---|---|
| 공격력 | 무력 | 통상공격 데미지 |
| 방어력 | 통솔력 | 받는 통상 데미지 |
| 정신력 | 지력 | 책략 위력 + 받는 책략 피해 감소 |
| 순발력 | 민첩성 | 방어(회피)확률 + 연속공격확률 + 명중률 |
| 사기 | (별도) | — |

- **부대 초기능력치(Lv0) = 장수능력 ÷ 2.** ← 우리 추출값(generals.json)이 표시 능력치의 절반인 이유. 런타임 이식 시 ×2가 아니라 **이 공식 자체를 쓰는 게 정석**(commanders는 장수능력 원값, 부대 스탯은 /2 + 성장).
- 무력/지력/통솔/민첩/운은 짝수. 열매(아이템) 복용 시 해당 능력 +2.

## 2. 성장 (레벨업)

```
부대능력(Lv) = floor(장수능력 / 2) + 등급계수(병과,스탯) × Lv
```
- 예: 민첩 90, 순발 등급 B, Lv10 → 45 + 3×10 = 75.
- 등급계수는 병과별·스탯별 S/A/B/C/D 등급에 따라 결정, **등급 간 0.5씩 차이**.
- 印綬(인수) 아이템으로 3단계 승급(병과 통일). 아군 가용 병과 13종 전부 3승급.
- 레벨캡 50 (영걸전 99에서 하향).

## 3. 데미지 공식

**통상 공격:**
```
데미지 = ((공격력 × 지형) − (방어력 × 지형)) ÷ 2 + 공격자레벨 + 25
        (최소 1)
```
- 병종 상성(유리/불리)으로 공격력/방어력 증감 적용.
- 지형: 성·관문·요새 점유 시 공·방 ×1.2.

**책략:**
```
데미지 = ((공격 정신력 − 방어 정신력) ÷ 3 + 공격자레벨 + 25) × 책략 고유 기본치
```
- 책략 기본치 = strategies.json `power` (초열6 < 업화10 < 화룡20 …).

**명중률:**
- 방어자 순발력 × 2 < 공격자 순발력 → 100%.
- 그 외 순발력 비율 기반(복잡식). + 연속공격 확률 별도.

## 4. 병과 등급 체계

병과(27종, 아군 13종)마다 5스탯(공격/정신/방어/순발/사기)에 S/A/B/C/D 등급.
예: 서량기병 공S 정C 방S 순B 사B / 제독(도위) 공A 정S 방A 순B 사B.

### classes.json 매핑 — ✅ 확정 (2026-06-14, → sosoden-class-grades.md)
`terrain_c2_6`(5값) = 병과 등급표. **인코딩 4=S/3=A/2=B/1=C, 열 순서 [공·방·정·순·사]** (우만위키 미러 교차검증, 아래 가설이 전 병과 일치). 등급계수 구간표도 확보 → W3 이식 대기.

(아래는 확정 전 가설 기록)
`classes.json`의 `terrain_c2_6`(5값, 1~4)가 이 5등급일 가능성:
- 서량기병(39) `[4,4,1,2,2]`를 **[공,방,정,순,사]·S=4/A=3/B=2/C=1**로 읽으면 = 공S 방S 정C 순B 사B → **알려진 등급과 일치**.
- 단 궁병 등 일부 자리가 안 맞아 순서/인코딩 미확정. **전체 병과 등급표(namu "조조전/병과") 대조 필요** — 봇차단이라 직접 fetch 불가.
- ⚠️ namu 병과 등급표 + 등급별 상승치표 이미지를 확보하면 c2_6 인코딩·등급계수를 확정 → ② 구현 가능.

## 5. 영걸전→조조전 이식 — **✅ 통상전 공식·병력 스케일 이식 완료 (2026-06-13)**

> combat.ts가 조조전 공식으로 교체됨: 부대능력=floor(장수능력/2)+성장(LV_GROWTH 잠정 3),
> 데미지=(공격력−방어력×상성)/2+Lv+25, 병력 조조전 스케일(~100~150). 책략 MVP(화계)도 구현.
> 미완: 상태이상·책략 전 카테고리·승급. (등급계수 ✅ W3, **명중률(순발)·전투특성 ✅ 2026-06-16 시드확률** — sosoden-port-status.md §7.) 아래 표는 이식 전후 대조 기록.

### (참고) 영걸전 vs 조조전 대조

| 항목 | 영걸전 (현 combat.ts) | 조조전 (목표) |
|---|---|---|
| 공격력 | 병종 baseAtk + 사기 + 보정무력, ×(10+Lv)/10 | floor(무력/2)+등급계수×Lv (병종 baseAtk 없음) |
| 데미지 | atk − def×상성/2, ×(1−guard) | (atk×지형 − def×지형)/2 + Lv + 25 |
| 레벨 보정 | ×(10+Lv)/10 곱 | + Lv (가산) + 25 상수 |
| 지형 | 방어자 guard 감산(0~0.3) | 점유지 공·방 ×1.2 |
| 책략 | (현재 미구현) | (공정신−방정신)/3+Lv+25 × 기본치 |
| 명중 | 100% 결정론 | 순발력 비율(회피·연속공격 포함) |

→ ②(이식) 시 combat.ts·UnitState·schemas(baseAtk/baseDef 제거, 등급/순발/정신/사기 추가) 재설계 필요. 명중률 도입은 RNG 결정론(시드) 계약 유지 전제.
