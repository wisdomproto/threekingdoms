## 공통 시스템 메모 (전투 전반)

- **승리조건 패턴**: ① N턴 내 특정 보스 격멸/퇴각, ② 특정 아군(주로 유비)이 지정 지점(성문·성채·군량고) 도착, ③ 특정 NPC 설득 완료, ④ 적 전멸. 다수 전투가 "N턴 내" 제한과 함께 걸린다(30턴 표준, 공성·대형맵은 40~45턴).
- **적 회복 AI**: 적은 병력 1/3 이하가 되면 마을로 들어가 회복한다 → 경험치 파밍 시 마을로 못 들어가게 막거나, 마을 안에서 처치하도록 유도하는 공략이 반복 등장.
- **보스 정지 패턴**: 다수 보스가 일정 턴까지 정지해 있다가(예: 사수관 화웅 = 무이동, 신도 순우경 = 14턴, 호로관 여포 = 18턴) 이후 행동 개시. → 우리 페이싱 커브(§11)에서 "보스 활성화 턴"을 데이터화할 근거.
- **증원군(援軍)**: 아군 NPC 증원(예: 계교 조운/공손찬)과 적 증원(예: 회남 6턴 여포군, 하구 5턴 3부대) 둘 다 존재. 등장은 턴 트리거 또는 위치 트리거.
- **보물 획득 3문법**(우리 §10과 일치): ① 보물창고(맵 위 지점 도달/점령), ② 선택지/화친 이벤트(특정 적 설득·화친), ③ 특정 적 일기토 격파.
- **분기**: 초반부터 A/B 갈림길이 잦다(신도 vs 광천, 거록 vs 청하, 팽성 vs 하구 등). 각 분기는 합류 장수·보물·도구상 개방 여부가 달라진다 → 우리 가상모드 분기 설계의 원형.

  - 적은 병력 1/3 이하 시 마을로 들어가 회복 → 고레벨 적을 마을 밖에서 잡아 경험치 극대화.
  - 초열서(공격 책략서) 사용으로 데미지 120~160 추가 경험치 획득.

- **일기토**: 장비 vs 여포 (장비 일기토 진행 시 보스 경험치 획득).
- **이벤트/분기**: 전투 후 **간옹 합류**(궁병 / LV3 / 능력치 42·74·36). 이후 분기: **신도(번궁 합류)** vs **광천(곽적·한영 합류)**.
