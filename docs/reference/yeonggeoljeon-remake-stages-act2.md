> **⚠️ 주의 4가지.**
> 1. **팬 리메이크 + 가상 모드다.** 합류 장수·전투 무대·결과가 사서와 크게 다르다(예: S29~30 낙성에서 방통 전사 미발생, S42 완성에 하후돈·우금 동시 등장, 최종전 무대가 적벽이 아닌 업). 사실(史實) 고증 자료로 쓰지 말 것 — **"가상 분기 콘텐츠 설계 참고용"**이다.
> 2. **영상이 시간순이 아니다.** 콜드오픈·회상·다음 화 예고·컷신 몽타주가 프레임 순서에 섞여 있다. 분류는 각 원시 노트가 **상태띠 스테이지명**으로 판별한 결과를 따른다.

> 4. **방송 오버레이 제외.** 화면 상단의 후원/채팅("○○님 1,000원 감사합니다" 등)은 게임 텍스트가 아니며 분석에서 제외했다.

> **신뢰도 태그.** `[영상]` = 영상에서 직접 확인 / `(추정)` = 정황·대사로 역산 / `(판독불가)` = 프레임 부족·화면 누락으로 미확인.

## 1. 스테이지별 상세

### S28. 공안의 전투

## 2. 공통 패턴·설계 시사점 (2부 특유)

1. **가상 통일 서사 = what-if 분기의 완결형.** 사실 라인을 뒤집어(관우 생존·조조 격파) **「한조재흥」 통일 엔딩**까지 27스테이지를 끌고 갔다. 사실 모드 종료 후 "유비가 천하를 통일했다면"을 별도 챕터로 전개한 실작례 — 우리 §5·§15 가상 분기의 직접 선례.
2. **신규 합류 장수·병종 대거 확장.** 1부 로스터 위에 **마초·황충·위연·방통·법정·강유·감녕·사마가·관흥·장포·엄안·이엄·마대·마량·마속·장완·서서·장료** 합류. 병종도 친위대·무도가대·환술사·주술사·대수적(해적계)·기습기병·벽력차·연노병·이민족(만병계)·근위병·도기병 등 후반 고급 병종이 다수 등장.
3. **세대 교체 서사.** S39 이후 관우·장비 사후를 건너뛰고 **관흥·장포**(2세대)가 등장("아버지를 닮아 건장하구나") — 장기 캠페인의 세대 교체 연출.
4. **후반 = 대형 공성·관문 돌파 다발.** 가맹관(S31)·와구관2(S34)·양평관(S38)·양양2(S40)·완성(S42)·진창(S43)·장안(S44)·허창(S45)·업1/2/3(S46~48)이 전부 성벽/관문/요새 미로 돌파전. **공성 페이싱**(다리 도하→외벽→내벽→본진)이 후반의 지배적 템플릿.
5. **광역 책략(화공/수공) 빈출.** 화공=낙성2(S30)·맥성(S39, 확산형)·허창(S45, 화룡 트랩)·업2/3(S47/48). 수공/범람=신야(S41)·진창(S43)·업3(S48). 제갈량은 4속성×4단계 풀 책략 세트 보유 — **후반 전투의 핵심 변수가 광역 속성 책략**으로 이동.
6. **동적 날씨·확산 지형.** 낙성(S29)·정군산(S35)·양평관(S38)에서 맑음→흐림→비 전환, 맥성(S39)에서 불길 턴별 확산 — 정적 타일이 아닌 **시간 진행형 지형 기믹**.
7. **협공·필살 커맨드 실사용.** 신야(S41)·업1(S46) 행동 메뉴에 **합공/협공·필살(결정타)** 노출·사용 — 우리 §8/§9 합공·필살 설계의 원작 UX 근거.
8. **일기토는 여전히 스토리 비트의 정점.** 확정 일기토: 관우vs여몽(S28)·뇌동(우군)vs장합(S33)·황충vs하후연(S35, 노장 명장면)·조인vs아군(S40)·하후돈vs관우(S42). **적장이 먼저 도발하는 일기토**(조인·하후돈)가 2부의 새 패턴.
9. **[전략조건]→보물 보상 시스템.** S28(성채 4점령+관우vs여몽 일기토)·S38(전략조건 달성→봉황깃옷)에서 **숨은 조건 달성 시 보물 수여**가 명시 — 초선이 사전 예고하고 사후 수여하는 UX. 우리 §10 보물 3문법/§11 S랭크 레이어 설계와 직결.
