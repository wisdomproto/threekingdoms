# 영걸전 한글 리메이크판 — 마스터 스테이지 레퍼런스 (27스테이지)

> **형제 문서.** 2부(가상 입촉~통일편, S28~S48) = [`yeonggeoljeon-remake-stages-act2.md`](yeonggeoljeon-remake-stages-act2.md).

> **목적.** 영걸전 한글 팬 리메이크판 플레이 영상을 스테이지별로 분석한 원시 노트를 하나로 종합한 문서. 우리 팀이 (a) 각 스테이지 **맵을 재현(베끼기)** 하고, (b) **시나리오·이벤트·일기토·전략조건**을 참고하기 위한 단일 자료다.
>
> **출처.**
> - **스테이지 1~2 (사수관·호로관):** 63분 단일 플레이 영상의 세그먼트 분석 `analysis/seg_01.md ~ seg_10.md` (프레임 g0001~g0383, 10초/프레임).
> - **스테이지 3~27:** 축약본 영상 시리즈 `analysis/series_02.md ~ series_16.md` (영상 1편이 1~3 스테이지를 담음, 프레임 표기 `gNNNN`).
> - **엔진 확인:** 메뉴 로고 `三國志英傑伝` + "Copyright 2009–2011 KOEI & waddaa" (series_15). = **코에이 영걸전 기반 팬 리메이크/모드**. 병종·일기토·턴 문법은 영걸전 원작 그대로.
>
> **⚠️ 주의 3가지.**
> 1. **팬 리메이크다.** 수치·연출·전략조건·보물이 코에이 정본(DOS 영걸전)과 다를 수 있다. 영상 내 NPC '허자장'이 직접 "기존 도스 영걸전 번역은 발로 해서…", "기존 영걸전과 다르게 병력·책략치 전부 회복되어 있다" 등 원작과의 차이를 자주 언급한다.
> 2. **영상이 시간순이 아니다.** 콜드오픈(예고 컷)·회상·다음 화 예고·컷신 몽타주가 프레임 순서에 섞여 있다. 본 문서는 각 원시 노트가 **상태띠 스테이지명**으로 분류한 결과를 따른다.
> 3. **리메이크 스테이지 구성 = 원작 영걸전 유비전**이다. 우리 `CLAUDE.md §5`의 27스테이지 재구성과는 챕터 경계·스테이지 선정이 다르다(특히 리메이크는 **형주 4군 평정=영릉·장사**로 끝나고, 우리는 적벽으로 간다). 대응표는 §5 참조.
>
> **신뢰도 태그.** `[영상]` = 영상에서 직접 확인된 사실 / `(추정)` = 정황·대사로 역산 / `(판독불가)` = 프레임 부족·화면 누락으로 미확인.
>
> **맵 분업 원칙.**
> - **지형 기하학(격자 크기·강·산 윤곽) = HEXZMAP(정밀, 원천).** `yeonggeoljeon-data.md §8-2` 추출 크기를 병기한다.
> - **배치·이벤트·목표·시나리오·일기토 = 영상(유일 소스).** 미니맵 프레임을 그대로 격자화하면 강 곡선·다리·진영 방위가 잡힌다.

## 1. 스테이지별 상세

### S01. 사수관의 전투
- **맵.** 가로 긴 평지 회전맵. 좌측 흙길/관문, 중앙 초원, 우측 강·숲. 지형 타일: 평지·숲·강(반짝이 텍스처)·목책·텐트/막사·병영. 우군 NPC 참전형. **HEXZMAP 56×32.** 최대 20턴. 베끼기 프레임: (seg) 전투 미니맵 g0097 이후(아군 파란점·적 붉은점·뷰포트 흰 박스). `[영상]`
- **초기 배치.** 아군 좌측 진입: 유비(의용장)·관우(군신)·장비(투신) 삼형제 + 보병. **우군=공손찬(경기병 Lv5)** 동맹 참전 — 진영 3종(아군/우군/적군) 확인. 적 보스 **화웅**(경기병 Lv5, 병영 110%), 적측 네임드 조잠·이숙(궁병). 황건적풍 청색 보졸 다수. `[영상]`
- **이벤트·목표.** 패배 조건 명시(컷신 허자장): **유비·관우·장비 중 한 명이라도 퇴각하면 패배.** 튜토리얼성 조언(병종 상성·레벨 안내) 동반. 회복 오브젝트(흰 슬라임형) 맵 배치. **일기토(스토리/즉사): 관우 vs 화웅** — VS 카드→연못 승부 컷신→즉사 대사 컷신. 결산 보상금 800. 격파 후 보물상자 다수 노출. `[영상]`
- **시나리오.** 반동탁연합. 원술이 무명의 유비 삼형제를 멸시하나, 유비가 "화웅을 못 베면 목을 내놓겠다"며 출전 자청. 관우가 화웅을 단칼에 베어 무명을 떨침. (도입부에 일기토 시스템 튜토리얼이 '제갈량 초상 차용 메타 화자'로 안내됨 — 시대상 미등장 인물이나 시스템 설명용.) `[영상]`

### S02. 호로관의 전투
- **맵.** 세로로 긴 맵. 전반=강+나무다리(부교, 폭 2칸) 병목의 평지 본진, 후반=북쪽 관문/성벽 지대로 전선 이동(성내 110% 지형, 돌계단 성문 + 강 위 목조 다리 병목). 성내 가시/녹각 저지물 패턴. **HEXZMAP 32×48.** 최대 20턴. 베끼기 프레임: (seg) g0228 이후 미니맵(좌하단). `[영상]`
- **초기 배치.** 아군 강 남안에서 출발, 다리 건너 북상. 유비·관우·장비 + 보병·기병. **우군=공손찬** 계속 참전. 적 보스 **여포**(무신 Lv7, HP155). 적 네임드 **장료**(청주병 Lv3), 송헌(궁병), 이각(경보병). 청색 졸병 대군. `[영상]`
- **이벤트·목표.** **승리조건 오버레이: "여포를 퇴각시켜라!" / 제한 턴 20.** 공략 핵심 컷신(허자장): ①여포는 **직접 공격으로 퇴각 거의 불가** → **장비와의 일기토로 유도**가 정답. ②**12턴 아군 차례에 여포 이동범위에서 멀리 떨어질 것.** ③장비vs여포 일기토를 보면 보물 **[쌀]** 지급. **일기토(스토리, 즉사 아님): 여포 vs 장비** — VS 카드→흑백 동심원 결투 컷신, **여포 우세 결말**(장비 패배+레벨업). 여포는 적토마로 도주(관우 추격 실패). 결산 보상금 1000. 회복/드랍 오브젝트 맵 배치. `[영상]`

## 2. 공통 게임플레이 패턴 (★ 설계 시사점)

원시 노트 전반에서 반복 검출된 패턴. 우리 `CLAUDE.md §11`(스테이지 생성기·이벤트 트리거·전략조건)·`§8~9`(스킬·일기토)·`§10`(보물) 설계의 직접 입력값.

1. **전략조건(서브 클리어 조건) = 보물/보상 게이트.** 거의 모든 스테이지가 "특정 일기토를 본다 / 특정 순서로 일기토를 발동한다 / 특정 인물을 특정 위치로 이동시킨다"를 **전략조건**으로 걸고, 달성 시 NPC 허자장이 보물·보상금을 지급. 예: 태산(일기토+대화 2개), 회남(일기토+성채 피신 2개), 광릉([콩주머니]), 박망파([가죽방패]), 장판파I([기마갑옷])·II([제갈건], 2목표 동시), 강하(일기토 **순서**→[적로]), 영천(보상금 2800). **일부 스테이지는 명시적으로 "전략조건 없음"**(서주II·박망파·고성)으로 자유 클리어를 선언 — 난이도 리듬 조절 장치. → **우리 S랭크/챌린지 레이어(§11)와 직결.** 트리거를 "일기토 발동 / 인물 도달 / 발동 순서"로 데이터화.

2. **일기토 = 발동 조건 자체가 공략 요소.** 일반 직접공격으로는 보스 격파가 거의 불가능한 케이스(호로관 여포 — "직접 공격으론 퇴각 거의 불가, 장비 일기토로 유도"), 발동 순서가 보물을 가르는 케이스(강하), 보스 처치를 일기토로 하는 케이스(청하 국의·팽성 조하·계교 문추 등 다수). **스토리 일기토는 즉사 허용**(사수관 관우vs화웅), **일반 일기토는 즉사 없음·승패 무관 레벨업**(호로관 장비, 강하). → 우리 `§9`의 "스토리=즉사 전용 / 일반=즉사 없음" 분리와 완전 일치.

3. **보물 "택1" 분기.** 광천 "두 개의 보물고 중 한 군데에서만 획득" — 2회차 동력·도감 떡밥(우리 `§10` 보물 도감과 일치). 청하 등에도 동형 거점 존재.

4. **목표가 섬멸이 아닌 스테이지가 다수.**
   - **탈출/이탈전:** 여남(조인 격파 금지·전멸 금지하고 빠져나가기), 연주(조운 탈출), 서주II(패주 이벤트 전투).
   - **호위/도하전:** 장판파I·II(백성 전멸=패배, 강 서안 탈출지점 도달=승리).
   - **시간 압박 거점 점령:** 강릉(주유 도착 전 점령).
   - **방어/구원전:** 서주(서문 돌파해 도겸 접촉), 북해(공융 구원).
   - → 우리 `§11` 지형 문법 템플릿("관문 돌파/협공 방어/탈출/화공 이벤트")의 실증 카탈로그.

5. **우군(동맹 NPC) 활용 = 제3진영.** 진영 3종(아군/우군/적군) 시스템. 공손찬(사수관·호로관·계교), 도겸(서주), 조조군(하비), 유벽(여남), 손권·주유 관전(강하). 우군은 지형보정·교전을 공유하나 직접 조작 불가. → 우리 엔진의 faction 모델에 "ally/neutral" 추가 근거.

6. **증원/적대 전환 트리거.** 태사자(서주 턴6), 하후돈 청주군 대증원(서주II, 차주 퇴각 후 전세 역전), 오군 적대 전환(강릉, 유비군 성 공격 시작 시), 한영 2장수 투항(광천), 조조군 우군 증원(하비). → 트리거를 턴/이벤트(특정 행동)에 바인딩.

7. **화공 트리거 = 유인→점화→혼란→섬멸 4단.** 박망파 정본 사례(제갈량 유인 지시→조운 미끼→유봉·관평 점화→불 격자 확산+적 ? 마크 혼란→섬멸). 강 위 "푸른 화염" 특수 타일(장사). → 우리 `§3-1` VFX 레이어 + `§11` "화공 이벤트" 템플릿 구현 스펙.

8. **야수/특수 병종.** 맹호대(호랑이)·곰부대 유닛이 아군/적 양쪽에 빈출(태산·하구·북해·하비·여남·박망파·영릉). 손상향=호랑이 동반 여성 유닛(고성). → 조조전 특수병종(맹호대 등)과 일치(`§7`).

9. **분기 선택(스테이지/경로 택1).** 광천↔신도성, 청하↔거록, 양양↔강하, 초가집 설득/경청. 선택 안 한 분기는 미플레이. → 우리 가상 모드 분기(`§5`)·회차 동력.

10. **반전·재회·합류 서사 비트.** 고성(산적 두목=장비 반전), 연주·고성(조운·장비 재회), 영천(관평·주창 합류), 장사(위연 합류·황충 설득 아크), 서서 합류→이탈("이탈 장수 자원 환급"=우리 `§6`). → 로스터 합류/이탈 이벤트 데이터화.

11. **NPC 메타 가이드(허자장).** 전 스테이지 공통으로 전략조건·공략 힌트·원작 비교를 코멘트하는 비전투 NPC. → 우리 튜토리얼/힌트 시스템의 단일 화자 패턴 참고(원작엔 없던, 리메이크 자체 추가 요소이므로 채택 여부는 별도 판단).

12. **결산은 미니멀.** "작전종료 + 보상금 숫자"만 맵 위 오버레이(별 평가·경험치 바·상자 개봉 없음). → 우리 `§12` 도파민 결산("별 평가→상자→레벨업 풀 연출")은 **원작/리메이크보다 강화하는 방향**임을 재확인(차별화 포인트).

**핵심 차이.**
- **엔딩 분기:** 리메이크는 **형주 4군 평정(영릉·장사)으로 1부 종결** — 적벽은 **컷신/내레이션으로만** 처리하고 플레이 맵이 없다(강릉 영상에서 확인). 우리 `§5`는 **적벽대전~화용도(23~27)를 플레이 클라이맥스**로 두는 정반대 설계.
- **챕터 구성:** 리메이크에는 우리 황건적의 난 챕터(탁군·광종·장각)와 적벽 5스테이지가 통째로 없고, 대신 평원-공손찬-계교 아크(광천·청하·계교·북해)·산적 3연전(태산·하구·팽성)·원소 의탁기(연주·고성)·형주 남부 평정(영릉·장사)이 들어 있다.
- **솔로 스테이지 부재:** 우리 `§5`의 "조운 단기필마(20)"·"장비 단신 방어(21)" 같은 솔로 명물이 리메이크엔 없고, 장판파를 **군세 호위전 + 조운/하후은 일기토**로 구현한다. 솔로 연출을 원하면 우리가 별도 설계해야 함.

## 부록: 화면/HUD 문법 요약 (재현 참고)

- **시점:** 톱다운 직교(쿼터뷰 아님, 원작 영걸전 그대로). 도트 스프라이트 유닛 + painted/타일 배경. 해상도 640×400급.
- **상태띠:** 좌=스테이지명(파란 음각) / 우=「턴 수 N/최대」(붉은 라벨). 막간 화면에선 좌=장 제목/우=위치명 + 진행도 게이지바.
- **미니맵(「전장 지도」):** 우상단 또는 좌하단 별도 창. **아군=파란 점, 적군=빨간 점, 우군/중립=주황 점**(영상별 색 배치가 섞이는 경우 위치로 판별), 흰 사각=뷰포트.
- **하이라이트:** 이동 범위=반투명 파란(때로 녹색) 칸 / 공격 범위=빨간 테두리 칸 / 커서·선택=흰 사각.
- **유닛 호버 팝업:** 이름(아군 파랑·적 빨강)+병종+Lv, HP(하트/녹)·MP(깃털/노랑)·SP(0/255, 사기 추정), 진영+지형명+보정%(평지100~110%/병영110%/성내110%/보물고100%/가옥110% 등). 공격 대상 호버 시 "피해 N / 명중 N%" 예측 행.
- **장수 정보 패널:** 탭 5종(능력/장비/책략/장수열전/부대특성). 부대특성=S~C 능력 등급+공격범위 5×5 그리드(근접=인접 8칸/십자, 궁병=확장 십자). 장수열전=무력/민첩/지력/행운/통솔 + 출진수/퇴각수.
- **커맨드 메뉴 8항목:** 공격/책략/도구/교환/협공/필살/대기/취소(조건 미충족=회색 비활성).
- **일기토:** VS 카드 인트로 → 흑백 동심원(수상/안개) 결투 컷신 → 양측 초상+대사. 무관=도트 스프라이트 합 연출(영상 아님). 공격/돌격/방어/반격 4카드(HP·사기 N/5) 입력형도 존재(튜토용).
- **결산:** "작전종료 + 보상금 N"만 맵 위 오버레이(미니멀).
- **장비:** 무기/방어/보조 3슬롯, 무기·방어구 Lv+Exp 성장. 특수효과(치명일격공격/범위공격 이격/책략모방/인도공격/선제공격 등). 상점 막간 4버튼(출진/장비/구입/매각). 세이브 10슬롯(진행 지점 자동 라벨).
