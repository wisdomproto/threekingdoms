에셋 프롬프트 보드 v2.0

v1 캐릭터 비주얼 = ① SD 포즈시트(idle/move/attack 한 장 → 📤 넘기기 → Claude 컷 → sprites/{id}) + ② 초상시트(여러 명 한 장 → 📤 넘기기 → Claude 슬라이스 → ui/portraits/{이름}) + ③ 시나리오 씬. ¾ 좌향·투명배경, 뒤모습 선택(front 폴백). 보조: UI·이펙트·지형. ⏸ 보류(v1.5+/막바지): 전신레퍼·무기스킨·일기토키프레임·리그. 컷·슬라이스는 Claude가 처리. 화풍: 수묵담채 (확정)
사용법 ① 카드의 [프롬프트 복사] → 생성기에 붙여넣어 생성 ② 결과 이미지를 복사한 뒤 카드의 점선 영역 클릭 + Ctrl+V (드래그&드롭/파일선택 가능) ③ 잘 나온 장은 ★픽. 한 장에 여러 에셋이 그리드로 나오는 시트 방식이라 — 셀 일부가 깨져도 살릴 셀이 있으면 픽하고, 깨진 셀만 따로 메모 주면 됩니다. 슬라이스(셀 분리·배경 제거)는 Claude가 합니다.
저장은 이 브라우저 IndexedDB에 자동 보존(다시 열어도 유지). ⬇ 버튼은 연결된 폴더(상단에서 1회 연결, docs\art 선택)에 바로 파일로 저장됩니다 — 픽한 컷은 ⬇ 눌러두면 Claude가 그 폴더에서 슬라이스 작업을 이어받습니다. 캐릭터 후속 생성(A-2~A-4)에는 픽된 A-1 시트를 참조 이미지로 첨부하세요.
✂ 파츠 컷 박스를 끌어 각 파츠 영역에 맞추세요 (모서리=크기). 투명 배경이 그대로 보존됩니다.
