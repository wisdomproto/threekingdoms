스테이지별 배경 = 블록아웃 조각 → img2img → painted 채우기 → Claude가 stitch
사용법 ① 각 카드의 블록아웃을 Gemini에 img2img 입력으로 넣고 아래 공통 프롬프트로 생성 ② 결과를 카드 우측 점선 영역에 Ctrl+V(드래그&드롭/파일선택) — 한 조각당 1장 ③ 한 스테이지 조각을 다 채우면 하단 stitch 명령을 Claude에게 주면 합쳐서 배경 교체.
저장 폴더를 docs\art\chunks로 연결하면 붙여넣는 즉시 painted_{stage}_r#_c# 파일로 자동 저장돼 바로 stitch 가능.

img2img 공통 프롬프트 6장 모두 동일 — 첫 조각 잘 나오면 나머지 생성에 스타일 참조로 첨부
